# ainews-reports

AI 뉴스레터 한국어 번역 아카이브

[ainews](https://buttondown.com/ainews) 및 [news.smol.ai](https://news.smol.ai) 뉴스레터를 자동으로 한국어 번역·요약하여 보관합니다.

## 구조

```
reports/
├── ainews/                # ainews 뉴스레터 번역
│   └── YYYYMMDD_*.md
└── smolai/                # news.smol.ai 뉴스레터 번역
    └── YYYY-MM/
        ├── *slug*.md              # 전체 번역
        ├── *slug*_summary.md      # 요약 (Markdown)
        └── *slug*_summary.html    # 요약 (HTML)
```
