Autoresearch: Sparks of Recursive Self Improvement - 요약

원문: https://www.latent.space/p/ainews-autoresearch-sparks-of-recursive... 번역일: 2026-03-10 07:28

다음은 AI 뉴스레터에서 추출한 핵심 신규 소식 요약 브리핑입니다.

🔥 주요 뉴스

  • [LLM의 자율적 자기 개선 시연] — Andrej Karpathy는 에이전트 기반 연구 루프를 통해 nanochat 학습에서 “Time to GPT-2”를 약 11% (2.02시간에서 1.80시간으로) 단축했다고 보고했습니다. 이는 LLM이 스스로 학습 레시피를 체계적으로 개선할 수 있음을 보여주는 중요한 진전입니다.
  • [Anthropic, 멀티 에이전트 PR 코드 리뷰 출시] — Anthropic이 Claude Code를 활용한 멀티 에이전트 PR 리뷰 도구를 출시했습니다. 이 도구는 의미 있는 PR 댓글 비율을 16%에서 54%로 증가시키고, 잘못된 발견은 1% 미만으로 유지한다고 주장합니다.
  • [OpenAI, Promptfoo 인수] — OpenAI가 오픈소스 에이전틱 보안 테스트 및 평가 도구인 Promptfoo를 인수했습니다. Promptfoo는 오픈소스로 유지되며, OpenAI의 에이전트 보안 테스트 및 평가 역량을 강화할 예정입니다.
  • [Figure Helix 02, 자율 주택 청소 데모 공개] — Brett Adcock이 Figure Helix 02 로봇의 완전 자율 거실 청소 데모를 공개하며, 전신 로봇이 엔드투엔드 가사 작업을 수행하는 중요한 이정표를 제시했습니다.
  • [Anthropic, 미 국방부 상대 소송 제기] — Anthropic이 대량 감시 및 자율 무기에 대한 안전장치 제거를 거부한 후, 미 국방부에 의해 “공급망 위험”으로 분류되자 소송을 제기한 것으로 알려졌습니다. 이는 AI 기업의 정책 및 조달에 영향을 미칠 수 있는 주요 법적/정책적 사건입니다.

📊 모델 & 벤치마크

  • GPT-5.4 사용자 반응 및 성능 — ChatGPT의 GPT-5.4는 지시 준수 및 어려운 한국어 테이블 OCR과 같은 비전/OCR 작업에서 긍정적인 실제 사용 경험과 상당한 개선을 보였습니다.
  • Anthropic, 문서 분석 벤치마크 지배 — Document Arena 보고서에 따르면, 문서 분석 및 장문 추론 분야에서 상위 3개 모델이 Anthropic의 Opus 4.6, Sonnet 4.6, Opus 4.5로 나타났습니다.
  • Gemma 4 출시 루머 — 120B 전체/15B 활성 파라미터를 포함한 Gemma 4의 임박한 출시 루머가 돌고 있습니다.
  • Qwen3.5 로컬 실행 가이드 공개 — Unsloth는 ≤24GB RAM에서 작동하며 Unsloth를 사용하여 모델을 파인튜닝하는 에이전트를 시연하는 Qwen3.5 로컬 실행 가이드 및 파인튜닝 에이전트 워크플로우를 게시했습니다.
  • RoboMME, 로봇 메모리 벤치마크 등장 — RoboMME가 로봇 제너럴리스트 정책의 메모리 벤치마크로 새롭게 등장했습니다.

🛠️ 제품 & 도구

  • Cognition, Devin Review 출시 — Cognition이 URL 대체 방식의 무료 PR 리뷰 도구인 Devin Review를 출시했으며, 자동 수정 및 diff 기능을 포함합니다.
  • Perplexity “Computer” 기능 확장 — Perplexity는 “Perplexity Computer”에 Claude Code와 GitHub CLI를 추가하여 리포지토리 포크부터 PR 제출까지의 엔드투엔드 과정을 시연했습니다. 또한 Google/Meta Ads API 커넥터를 통해 자율적인 광고 캠페인 운영을 주장합니다.
  • Andrew Ng, Context Hub 출시 — Andrew Ng이 최신 API 문서를 가져와 API 환각을 줄이고 영구적인 주석 및 커뮤니티 공유를 지원하는 CLI 도구인 Context Hub를 출시했습니다.
  • LangSmith Agent Builder 기능 추가 — LangSmith는 멀티모달 평가자와 병렬 에이전트 작업을 관리하기 위한 Agent Builder 인박스를 추가했습니다.
  • Harbor, 엔드투엔드 컴퓨터 사용 평가 통합 — Harbor는 대규모 엔드투엔드 컴퓨터 사용 평가(Windows/Linux)를 통합하여 SFT/RL을 위한 궤적을 생성합니다.
  • Teleport, 에이전틱 아이덴티티 제안 — Teleport는 암호화된 아이덴티티, 최소 권한, MCP/도구 전반의 감사 추적을 위한 제어 플레인으로 “에이전틱 아이덴티티”를 제안했습니다.
  • LlamaIndex, Surreal Slides 시연 — LlamaIndex는 LlamaParse, SurrealDB, MCP 에이전트 인터페이스를 활용하여 슬라이드 덱 파싱 및 리트리벌(“Surreal Slides”)을 시연했습니다.
  • Hugging Face, LeRobot v0.5.0 업데이트 — Hugging Face의 로보틱스 스택인 LeRobot이 v0.5.0으로 주요 업데이트를 출시하며 Unitree G1 휴머노이드 지원, 새로운 정책, 실시간 청킹, 더 빠른 데이터셋, EnvHub/Isaac 통합, Python 3.12 + Transformers v5, 플러그인 시스템을 포함했습니다.

🔬 연구 & 논문

  • Karpathy, 오토리서치 성과 보고 — Andrej Karpathy는 에이전트 기반 연구 루프를 통해 nanochat 학습에서 “Time to GPT-2”를 약 11% (2.02시간에서 1.80시간으로) 단축했다고 보고했습니다. 이 루프는 정규화 스케일러, 어텐션 튜닝 등 학습 레시피 개선 사항을 체계적으로 발견합니다.
  • AgentIR, 추론 토큰 활용 성능 향상 — AgentIR은 에이전트의 “추론 토큰”을 신호로 사용하여 BrowseComp-Plus에서 기준선 대비 35%에서 67%로 성능 향상을 보고했습니다.
  • Memex(RL), 인덱싱된 경험 메모리 제안 — Memex(RL)는 컨텍스트 윈도우를 부풀리지 않고 장기적인 작업을 확장하기 위해 인덱싱된 경험 메모리 개념을 제안했습니다.
  • Databricks/DAIR, KARL 연구 발표 — Databricks/DAIR은 엔터프라이즈 검색 에이전트를 위한 멀티태스크 RL 학습 모델 KARL을 발표하며, 파레토 최적의 비용/레이턴시 품질 트레이드오프와 향상된 일반화를 주장합니다.
  • CosNet 아이디어 제시 — CosNet은 선형 레이어에 저랭크 비선형 잔여 함수를 추가하여 사전 학습에서 20% 이상의 실제 시간 속도 향상을 가져온다고 주장하는 아이디어를 제시했습니다.
