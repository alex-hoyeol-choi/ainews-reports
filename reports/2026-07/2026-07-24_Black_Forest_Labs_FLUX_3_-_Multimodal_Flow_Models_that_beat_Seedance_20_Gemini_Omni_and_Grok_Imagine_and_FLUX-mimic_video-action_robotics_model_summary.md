**[The Stack v3, 최대 규모 오픈 코드 데이터셋 공개]** — @anton_lozhkov가 현재까지 공개된 가장 큰 오픈 코드 데이터셋인 The Stack v3를 발표했습니다. 이 데이터셋은 114 TB의 원본 데이터, 2억 2,400만 개의 리포지토리, 440억 개의 파일, 770개 언어, 그리고 약 5조 개의 중복 제거/필터링된 토큰으로 구성되어 차세대 오픈 코드 모델 개발의 기반이 될 것입니다.

**[Etched, 3억 달러 시리즈 C 투자 유치]** — Etched가 인퍼런스 클러스터 생산 가속화를 위해 103억 달러 기업 가치로 3억 달러 규모의 시리즈 C 투자를 유치했습니다. 회사는 80,000평방피트/10 MW 규모의 시설을 개설하며 프론티어 모델 학습이 아닌 '전 세계의 인퍼런스 실행'에 집중할 계획입니다.

### 📊 모델 & 벤치마크
*   **Black Forest Labs FLUX 3 Video 출시:** 텍스트, 이미지, 비디오, 오디오, 키프레임 입력으로부터 비디오를 생성하며, 네이티브 오디오 생성, 다국어 대화, 다양한 시각 스타일 및 화면 비율, 에이전틱 체이닝을 지원하는 통합 멀티모달 모델입니다.
*   **Moonshot AI Kimi K3 출시:** 2.8조 파라미터의 오픈 웨이트 모델로, Aikido Security의 사이버 보안 벤치마크에서 GPT-5.6 Terra와 동등하고 GPT-5.6 Sol에 근접하는 성능을 보였습니다.
*   **Microsoft Fara1.5-27B 출시:** Microsoft Research AI Frontiers가 브라우저용 비전 전용 멀티모달 컴퓨터 사용 에이전트 Fara1.5-27B를 출시했습니다. Qwen3.5-27B를 기반으로 합성 작업/궤적 데이터로 지도 파인튜닝되었습니다.
*   **Alibaba Qwen-Audio-3.0-TTS 출시:** Flash 및 Plus 변형으로 16개 언어, 인라인 제어 태그, 자연어 스타일 조종, 노이즈 참조 강건성, 최대 3분 길이의 원패스 생성을 지원하며 Artificial Analysis TTS 리더보드에서 1위를 차지했다고 주장합니다.
*   **HuggingApps WordVoice TTS 강조:** 단어별로 지속 시간, 음량, 피치, 톤을 제어할 수 있는 소규모 TTS 모델로, 오디오 툴링을 위한 제어 표면 실험에 중점을 둡니다.
*   **OpenAI GPT-5.6 Sol 토큰 효율성:** OpenAI의 GPT-5.6 Sol이 현재 토큰 효율성 파레토 프론티어의 많은 부분을 지배하고 있습니다.
*   **CoreWeave MiniMax M3 속도 벤치마크:** MiniMax M3가 357 출력 토큰/초의 속도와 낮은 혼합 가격을 기록하며 제공자 속도 벤치마크를 게시했습니다.
*   **Frontier-Bench 출시:** @ryanmart3n이 코딩을 넘어 프론티어 에이전트 작업과 함께 발전하도록 고안된 지속적인 커뮤니티 벤치마크 Frontier-Bench를 출시했습니다.
*   **EnigmaEval 발표:** @CAIS가 Claude Fable 5와 GPT-5.6 Sol이 선두를 달리는 더 어려운 추론 벤치마크 EnigmaEval을 발표했습니다.

### 🛠️ 제품 & 도구
*   **OpenAI ChatGPT Voice 데스크톱 앱 출시:** Plus/Pro/Business/Edu/Enterprise 사용자를 위해 데스크톱 앱에 ChatGPT Voice가 출시되었으며, GPT-Live 기반으로 컴퓨터를 제어하고 ChatGPT Work 및 Codex 전반에서 작업을 조율할 수 있습니다.

*   **Andrew Ng OpenWorker 출시:** @AndrewYNg가 파일 및 업무용 도구를 위한 오픈소스, 모델 불가지론적 로컬 에이전트 OpenWorker를 출시했습니다.
*   **vLLM prime-rl 0.6.0 출시:** vLLM이 테라 규모 에이전틱 RL 인퍼런스 플러밍(FP8, 전문가 병렬 처리, 프리필/디코드 분리, KV 오프로드, 라우팅)을 설명했으며, 28개의 H200 노드에서 131k 시퀀스 길이의 SWE 작업에 GLM-5를 5분 미만 단계로 학습시키는 데 사용되었습니다.
*   **Hermes Profiles 출시:** @witcheer가 별도의 메모리, API 키, 세션, 게이트웨이, 내보내기/가져오기 경로를 가진 네임스페이스 에이전트 인스턴스인 Hermes Profiles를 출시했습니다.
*   **Microsoft VS Code 에이전트 앱 프로토콜 발표:** @davidfowl이 Microsoft의 VS Code 에이전트 앱의 기반이 되는 새로운 프로토콜을 발표했습니다.
