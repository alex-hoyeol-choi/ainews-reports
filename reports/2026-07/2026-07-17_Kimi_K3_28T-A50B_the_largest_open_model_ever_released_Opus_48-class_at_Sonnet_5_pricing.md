# AI 트위터 요약
Moonshot AI는 Kimi K3를 프론티어급 오픈 웨이트 모델로 출시했으며, 공식적으로 최고 성능의 클로즈드 모델에 근접하고 기존 오픈 경쟁 모델들을 능가한다고 주장했습니다.
*   Moonshot은 Kimi K3를 “Open Frontier Intelligence”로 공식 소개했으며, 2.8T의 총 파라미터, 1M 토큰 컨텍스트, 네이티브 멀티모달 입력, Kimi Delta Attention (KDA) 및 Attention Residuals를 특징으로 합니다. 이 모델은 Kimi.com, Kimi Work, Kimi Code 및 API에서 사용 가능하며, 2026년 7월 27일까지 오픈 웨이트를 공개할 예정이라고 밝혔습니다 @Kimi_Moonshot
*   Moonshot은 또한 장기적인 에이전틱 코딩 및 자체 진화 워크플로우를 중심으로 한 제품 포지셔닝과, 코드와 스크린샷을 반복하는 “vision in the loop” 코딩/게임 개발 워크플로우를 강조했습니다 @Kimi_Moonshot
*   공식 발표 전, 여러 계정에서 K3가 2.8T 파라미터라는 유출되거나 앱에서 얻은 세부 정보가 유포되었으며, 약속대로 웨이트가 출시된다면 역대 최대 규모의 오픈 웨이트 모델이 될 것이라고 언급했습니다 @scaling01, @scaling01, @eliebakouch
*   공식 Kimi 블로그는 나중에 공개되었으며, 주요 기술 자료로 널리 공유되었습니다 @Jianlin_S, @scaling01, @Yulun_Du
*   Moonshot의 자체 표현은 한계를 인정했습니다: 전반적으로 매우 경쟁력이 있음에도 불구하고, K3는 Claude Fable 5 및 GPT-5.6 Sol에 비해 여전히 “사용자 경험에서 눈에 띄는 격차”를 보인다고 했습니다 @scaling01
*   Arena는 Kimi K3가 Agent Arena, Text, Vision, Document 및 Frontend Code Arena에 진입했으며, 커뮤니티 평가가 뒤따를 것이라고 발표했습니다 @arena
*   Arena는 이어서 중요한 초기 결과를 보고했습니다: Kimi K3는 Frontend Code Arena에서 1679점으로 1위를 차지하며 Claude Fable 5를 능가했고, #18위(K2.6)에서 #1위로 뛰어올랐습니다. 7개 프론트엔드 도메인 중 6개에서 1위, 게이밍에서 2위를 기록했습니다 @arena
*   Arena는 나중에 K3가 Frontend Code Arena에서 76%의 쌍별 승률(pairwise win rate)을 기록했으며, Fable 5는 63%, GPT-5.6 Sol은 58%였다고 덧붙였습니다 @arena
*   Text Arena에서 K3는 1486점으로 9위에 올랐으며, #38위에서 도약한 결과입니다. 창작 글쓰기, 코딩, 지시 따르기에서 상위 10위권에 들었고, 여러 직업 분야에서 1위를 차지했습니다 @arena
*   Artificial Analysis는 K3를 AA Intelligence Index에서 57점으로 평가하는 독립적인 평가를 발표했으며, Opus 4.8 및 GPT-5.5와 비교할 만하지만, 전반적으로는 Fable 5 및 GPT-5.6 Sol에 여전히 뒤처진다고 언급했습니다 @ArtificialAnlys
*   AA는 또한 K3가 GDPval v2에서 1668 Elo, AutomationBench-AA에서 53% / 1위, AA-Briefcase에서 1547 Elo를 기록했다고 보고했습니다. 작업당 비용은 $0.94였으며, 전체 Intelligence Index 실행에서 K2.6보다 약 21% 적은 출력 토큰을 사용했습니다 @ArtificialAnlys
*   이번 출시는 엔지니어들과 모델 관찰자들로부터 즉각적인 강한 반응을 불러일으켰는데, 이들은 K3를 이전 DeepSeek의 순간들과 비견될 만한 오픈 모델의 이정표로 평가했습니다 @kimmonismus, @nrehiew_, @eliebakouch

## 기술 세부 정보
아키텍처 및 시스템 세부 정보
*   공식 사양: 2.8T 총 파라미터, 1M 컨텍스트, 네이티브 멀티모달 입력 (텍스트 + 이미지), 텍스트 출력, 7월 27일까지 오픈 웨이트 공개 @Kimi_Moonshot, @ArtificialAnlys
*   K3는 Kimi Delta Attention (KDA)을 사용하는데, Moonshot은 이를 통해 백만 토큰 컨텍스트에서 최대 6.3배 더 빠른 디코딩이 가능하다고 말합니다 @Kimi_Moonshot
*   또한 Attention Residuals (AttnRes)를 사용하는데, 이는 2% 미만의 추가 비용으로 약 25% 더 높은 학습 효율성을 제공한다고 주장됩니다 @Kimi_Moonshot
*   블로그의 커뮤니티 독자들은 추가적인 아키텍처 세부 정보를 강조했습니다: LatentMoE / Stable LatentMoE, 896개의 전문가 중 16개가 활성화되어 2% 미만의 활성화 비율을 의미합니다 @nrehiew_, @eliebakouch
*   블로그/보고서 토론에서 커뮤니티가 추가로 추출한 세부 정보: per-head Muon, QB 로드 밸런싱 / 퀀타일 로드 밸런싱, 그리고 SiTU (Sigmoid Tanh Unit)라는 새로운 활성화 함수 @eliebakouch
*   한 엔지니어는 이 아키텍처가 KDA + LatentMoE + AttnRes를 결합하면서 이전 Kimi 모델보다 2배 이상 스케일링되었다는 점에서 주목할 만하다고 언급했습니다 @teortaxesTex
*   KDA는 긴 인큐베이션 기간을 거쳤습니다: 디자인은 2025년 1월에 시작되어 프론티어 규모에 도달하기까지 약 1.5년이 걸렸다고 보고되었습니다 @zxytim
인퍼런스 및 서빙
*   K3 가격은 1M 입력 토큰당 $3, 1M 출력 토큰당 $15로 보고되었으며, 캐시된 입력은 90% 할인되어 1M당 $0.30입니다 @scaling01, @ArtificialAnlys
*   여러 게시자는 해당 가격을 Sonnet 5와 비교했으며, 일부는 Sonnet이 8월 말까지 일시적으로 더 저렴했지만 그 이후에는 가격이 더 밀접하게 일치한다고 언급했습니다 @kimmonismus
*   80% 입력 / 20% 출력의 혼합 추정치는 1M 토큰당 $5.40였으며, Opus 4.8의 $9 및 GPT-5.5의 $10와 비교됩니다 @jaminball
*   Artificial Analysis는 Intelligence Index 작업당 평균 비용을 $0.94로 추정했으며, GPT-5.6 Sol은 $1.04, Opus 4.8은 $1.80였습니다 @ArtificialAnlys
*   초기 라이브 서빙 관찰 결과: OpenRouter의 Moonshot API를 통해 약 28 tok/s를 기록했으며 @scaling01, 다른 관찰자는 26 tok/s를 보며 Opus보다 느리다고 평가하고 추측 디코딩(speculative decoding)이 아직 활성화되지 않았을 것이라고 추측했습니다 @nrehiew_, @nrehiew_
*   Moonshot의 블로그는 최고의 인퍼런스 효율성을 위해 64개 이상의 가속기가 있는 슈퍼노드 구성에 배포할 것을 권장한다고 보고되었습니다 @teortaxesTex
*   vLLM은 Moonshot이 KDA 프리픽스 캐싱 구현을 vLLM에 직접 기여했으며, 공식 출시를 위해 Day 0부터 지원이 가능하다고 밝혔습니다 @vllm_project
*   Moonshot의 KDA 기여는 KDA가 기존 프리픽스 캐싱의 가정을 깨뜨리기 때문에 중요하다고 언급되었으며, 따라서 업스트림 런타임 변경이 필요했습니다 @vllm_project
벤치마크 및 평가
*   Moonshot의 공식 벤치마킹 메시지는 다른 사람들에 의해 요약된 바에 따르면, 테스트된 모델 중 K3를 Claude Fable 5 및 GPT-5.6 Sol 다음으로, Claude Opus 4.8보다 앞선 위치에 두었습니다 @scaling01, @Yuchenj_UW

*   커뮤니티 게시물들은 또한 커널 최적화 작업에서 강력한 결과를 강조했으며, 일부는 K3가 특정 커널/코드젠 설정에서 Fable과 동등하거나 능가한다고 말했습니다 @nrehiew_, @scaling01
