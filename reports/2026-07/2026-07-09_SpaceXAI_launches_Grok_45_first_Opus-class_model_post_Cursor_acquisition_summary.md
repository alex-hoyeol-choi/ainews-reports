### 🔥 주요 뉴스
*   **[xAI, Grok 4.5 모델 출시]** — xAI가 Cursor와 협력하여 코딩 및 에이전트 워크플로우에 특화된 프론티어 모델 Grok 4.5를 공개했습니다. 이 모델은 Opus-클래스 성능을 목표로 하며, 속도와 비용 효율성을 핵심 강점으로 내세웁니다.
*   **[Grok 4.5, 1.5T 파라미터 및 고효율성 입증]** — Grok 4.5는 이전 Grok 4.3 대비 3배 커진 1.5T 파라미터 규모를 가지며, Artificial Analysis 벤치마크에서 프론티어급 성능과 함께 탁월한 비용/성능 효율성을 입증했습니다.

### 📊 모델 & 벤치마크
*   xAI가 코딩 및 에이전트 워크플로우에 특화된 새로운 프론티어 모델 Grok 4.5를 출시했습니다. 이 모델은 Opus 및 GPTs와 유사한 성능을 보이며, Grok 4.3 대비 3배 큰 1.5T 파라미터 규모를 가집니다.
*   Artificial Analysis Intelligence Index에서 54점으로 4위, GDPval-AA v2 Elo 1543점으로 4위를 기록했습니다. 특히 τ³-Banking 벤치마크에서 33%로 GPT-5.5(xhigh)의 31%를 능가했습니다.
*   Grok Build의 Artificial Analysis Coding Agent Index에서 76점을 기록하며 "Codex의 GPT-5.5와 동등한" 성능을 보였습니다.
*   Grok 4.5는 Opus 4.8 대비 Intelligence Index 작업당 평균 출력 토큰이 60% 이상 낮고, Coding Agent Index 작업당 총 토큰도 Claude Code의 Fable 5나 Codex의 GPT-5.5보다 현저히 낮아 뛰어난 효율성을 입증했습니다.

### 🛠️ 제품 & 도구
*   Grok 4.5는 Grok Build, API, Cursor에서 즉시 사용 가능하며, Hermes Agent, Portal, OpenRouter 및 Grok 구독을 통해 실시간으로 지원됩니다.
*   Cursor는 Grok 4.5를 자사 제품 내에서 "첫 주 동안 두 배 사용량"으로 사용할 수 있다고 발표했습니다.
