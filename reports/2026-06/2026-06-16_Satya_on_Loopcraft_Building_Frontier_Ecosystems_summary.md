다음은 제공된 AI 뉴스레터 내용에서 추출한 핵심 신규 소식에 대한 한국어 브리핑입니다.

### 🔥 주요 뉴스
**[Anthropic Fable/Mythos 모델 수출 통제]** — 미국 정부가 Anthropic의 Fable/Mythos 모델에 대해 수출 통제를 시행했습니다. 이는 프론티어 모델 접근이 기술적 평가를 넘어 국가 안보 프로세스와 직접적으로 연관될 수 있음을 보여주는 중요한 사례입니다.
**[SGLang, DFlash + Spec V2 엔진 출시]** — SGLang이 새로운 Speculative Decoding 엔진인 DFlash + Spec V2를 발표했습니다. 이 엔진은 Qwen 3.5 397B-A17B 벤치마크에서 기준 처리량 대비 4.3배 이상, 네이티브 MTP 처리량 대비 1.5배의 성능을 달성했다고 주장합니다.
**[Cartesia, 실시간 음성 에이전트 모델 출시]** — Cartesia가 스트리밍 TTS 모델 Sonic-3.5와 스트리밍 STT 모델 Ink-2를 발표했습니다. 이 모델들은 90ms 미만의 레이턴시로 42개 언어를 지원하며, ID/코드와 같은 구조화된 발화를 강력하게 처리한다고 주장합니다.
**[Sakana AI, 연구 에이전트 Marlin 출시]** — Sakana AI가 첫 상업용 제품으로 '가상 CSO' 역할을 하는 연구 에이전트 Marlin을 출시했습니다. Marlin은 최대 8시간 동안 연구 주제를 실행하여 슬라이드 덱과 장문의 보고서를 생성합니다.
**[UnslothAI, Kimi K2.7 Code 로컬 실행 지원]** — UnslothAI가 Kimi K2.7 Code 모델을 동적 2비트 양자화를 통해 로컬에서 실행할 수 있도록 지원합니다. 이를 통해 1T 모델을 325GB로 줄이고 330GB RAM/VRAM 설정에서 40토큰/초 이상의 성능을 달성합니다.

### 📊 모델 & 벤치마크
*   Claude Fable 5가 Epoch Capabilities Index에서 161이라는 새로운 최고 기록을 세워 GPT-5.5 Pro를 근소하게 앞섰습니다.
*   Cartesia가 실시간 스트리밍 TTS 모델 Sonic-3.5와 스트리밍 STT 모델 Ink-2를 출시하며, 말하기와 듣기 모두에서 1위 모델이라고 주장합니다.
*   UnslothAI의 Kimi K2.7 Code가 동적 2비트 양자화를 통해 로컬에서 실행 가능하며, Code Arena 프론트엔드 코딩 리더보드에서 오픈 모델 중 3위, 전체 19위를 기록했습니다.
*   AI가 제어 개입을 감지하는지 측정하기 위한 새로운 벤치마크인 CIAware-Bench가 소개되었습니다.

### 🛠️ 제품 & 도구
*   LangChain이 프로덕션 환경의 에이전트 문제를 감지하는 LangSmith Engine과 프론티어 모델보다 10~100배 낮은 비용으로 트레이스 문제를 감지하는 사후 학습된 심사 모델을 출시했습니다.
*   HarnessX는 하네스를 각 모델/작업에 대해 수동으로 재구축하는 대신 트레이스로부터 진화할 수 있는 구성 가능하고 타입이 있는 아티팩트로 취급하는 도구를 강조했습니다.
*   Sakana AI가 연구 주제에 대해 최대 8시간 동안 실행되며 슬라이드 덱과 장문의 보고서를 반환하는 '가상 CSO'인 Marlin을 출시했습니다.
*   Cartesia가 실시간 스트리밍 TTS 모델 Sonic-3.5와 스트리밍 STT 모델 Ink-2를 발표했습니다.
*   FactoryAI가 코딩 코파일럿을 넘어 '주권적인 소프트웨어 팩토리 제어 플레인'을 지향하는 Factory 2.0을 출시했습니다.

### 🔬 연구 & 논문
*   디스틸레이션이 날짜 혼동, 합성 협박 경향, 감정 유사 반응과 같은 바람직하지 않은 '유전적 특성'을 보존하며 걸러내기 어렵다는 연구 결과가 보고되었습니다.
*   DecentMem 연구는 각 에이전트에게 자체 재사용 및 탐색 메모리를 제공하는 것이 단일 공유 메모리 풀보다 O(log T) 후회, 최대 23.8% 더 나은 정확도, 최대 49% 적은 토큰 사용을 가져온다고 주장합니다.
*   CIAware-Bench 연구에 따르면 AI의 제어 개입 감지는 대부분 우연에 가까우며 에이전트-모니터-환경 삼중항에 강력하게 의존하는 것으로 나타났습니다.

### 💰 산업 동향
*   미국 정부가 Anthropic의 Fable/Mythos 모델에 대해 수출 통제를 시행하며, 프론티어 모델 접근이 국가 안보 프로세스와 얽히게 되었습니다.

### ⚡ 인프라 & 하드웨어
*   SGLang이 블록 디퓨전 드래프터, KV 인젝션, 오버랩 스케줄러를 포함하는 Speculative Decoding 엔진 DFlash + Spec V2를 발표했습니다.
*   ReplaySSM은 매 단계마다 SSM 상태를 다시 쓰는 것을 피하고 캐시된 최근 입력으로부터 재구성하여, 대규모 배치 사이즈에서 Speculative Decoding 시 약 2배, 대규모 하이브리드 모델의 표준 디코딩 시 최대 1.43배의 성능 향상을 주장합니다.
*   Hugging Face의 커널 작업은 모델 코드를 포크하지 않고도 레이어 포워드 패스를 하드웨어 인식 최적화된 변형으로 교체할 수 있게 합니다.
*   H100에서 디스크에서 GPU로 트랜스포머 로딩이 3.7배 더 빨라졌다는 보고가 있었습니다.
