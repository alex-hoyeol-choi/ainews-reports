### 🔥 주요 뉴스
**[Anthropic Claude 프로그래매틱 사용량 미터링 도입]** — Anthropic은 모든 Claude 구독에 구독 요금제 금액과 동일한 월별 API 토큰 크레딧을 부여하는 새로운 가격 정책을 발표했습니다. 이는 Claude.ai 및 Claude Code와 같은 Anthropic 소유 하네스 외의 프로그래매틱 사용량에 미터링을 적용하는 것으로, 파워 유저들에게는 구독 보조 하네스에 대한 주요 제한으로 해석됩니다.

**[Figure, 휴머노이드 로봇 Helix-02의 8시간 완전 자율 작업 시연]** — Figure는 휴머노이드 로봇 Helix-02가 소포 분류 작업에서 8시간 동안 완전 자율 교대 근무를 수행하는 라이브스트림을 공개했습니다. 이 로봇은 온디바이스 인퍼런스, 네트워크 플릿 조정, 자율 배터리 교체 및 자가 진단/유지 보수 기능을 갖추고 있습니다.
**[Nous Research, Token Superposition Training으로 사전 학습 속도 2-3배 향상]** — Nous Research는 Token Superposition Training(TST)이라는 새로운 사전 학습 방법을 발표했습니다. 이 방법은 동일한 FLOPs에서 인퍼런스 시 아키텍처 변경 없이 실제 학습 시간을 2~3배 단축하는 효과를 보였습니다.
**[LangChain, LangSmith Engine 등 에이전트 라이프사이클 인프라 대규모 출시]** — LangChain은 LangSmith Engine, SmithDB, Sandboxes, Managed Deep Agents 등 에이전트 라이프사이클 인프라를 대량으로 출시했습니다. 특히 SmithDB는 중첩되고 장기 실행되는 대규모 페이로드를 위한 관측 가능성 데이터베이스로, 주요 워크로드에서 12~15배 더 빠른 접근을 제공합니다.

### 📊 모델 & 벤치마크
*   Claude Mythos Preview는 AISI(영국 AI 보안 연구소)의 Cooling Tower를 포함한 엔드투엔드 사이버 레인지를 모두 해결한 최초의 모델이며, 연구소의 250만 토큰 제한 내에서 모든 작업을 완료했습니다.
*   Datology의 VLM 작업은 데이터 큐레이션만으로 2B 모델에서 20개 공개 VLM 벤치마크 전반에 걸쳐 +11.7점을 달성, InternVL3.5-2B를 약 10점 차이로 능가하며 학습 컴퓨팅을 약 17배 적게 사용했습니다.
*   영국 AI 보안 연구소는 프론티어 모델이 완료할 수 있는 사이버 작업의 길이가 몇 달마다 두 배로 증가하고 있으며, 최신 모델이 이전 추세를 능가한다고 보고했습니다.
*   Ramp 데이터에 따르면, 4월 기업 AI 채택에서 Anthropic이 34.4%, OpenAI가 32.3%를 차지하며 Anthropic이 처음으로 명백한 선두를 차지했습니다.

### 🛠️ 제품 & 도구
*   Cline은 재구축된 Cline SDK와 TUI, 에이전트 팀, 예약된 작업, 커넥터를 갖춘 CLI를 오픈소스화하여 맞춤형 코딩 에이전트를 위한 재사용 가능한 기반으로 포지셔닝했습니다.
*   LangChain은 LangSmith Engine, SmithDB, Sandboxes, Managed Deep Agents, LLM Gateway, Context Hub, Deep Agents 0.6 등 에이전트 라이프사이클 인프라를 대량으로 출시했습니다.
*   Notion의 External Agents API는 Claude, Codex, Cursor 등 서드파티 에이전트가 Notion 내에서 직접 작동할 수 있도록 지원합니다.
*   Cursor는 클론된 레포, 의존성, 버전 기록, 롤백, 범위가 지정된 이그레스, 격리된 시크릿을 포함한 완전히 구성된 개발 환경으로 클라우드 에이전트를 확장했습니다.
*   Duet Agent는 몇 주 또는 몇 달 동안 지속되는 작업을 위한 상태 머신 하네스를 제안하며, 압축을 부모/하위 에이전트 조정 및 메모리로 대체합니다.
*   LangChain의 오픈소스 업데이트는 스트리밍 타입 프로젝션, 체크포인트 저장, 코드 인터프리터, 하네스 프로필 및 모델별 튜닝을 추가했습니다.
*   Tabracadabra는 자동 완성에서 모든 텍스트 상자의 컨텍스트 인식 어시스턴트로 전환했습니다.
*   VS Code는 에이전트 창과 더 나은 다중 프로젝트 작업 검토 기능을 도입했습니다.
*   Perplexity는 VPC 수준 분리, 단기 프록시 토큰, 에이전트 작업 전 외부 콘텐츠 스캔을 포함하는 하드웨어 격리 샌드박스 아키텍처를 공개했습니다.
*   XBOW는 "토큰 단위로 전례 없는 정밀도"로 파트너 사용을 통해 몇 주 만에 수천 개의 높음/치명적 취약점을 발견했다고 보고되었습니다.
*   Victor Mustar는 llama-eval을 더 비교 가능한 llama.cpp 커뮤니티 평가를 향한 한 단계로 지적했습니다.

### 🔬 연구 & 논문
*   Nous Research의 Token Superposition Training은 사전 학습의 초기 단계를 수정하여 모델이 표준 다음 토큰 예측으로 돌아가기 전에 연속적인 토큰 묶음을 읽고 예측하도록 하여 2-3배의 실제 시간 속도 향상을 보고했습니다.
*   Jonas Geiping 외 연구진은 현재 메시지 기반/채팅 학습이 에이전트를 단일 스트림으로 과도하게 제한한다고 주장하며, 더 낮은 레이턴시, 더 깔끔한 관심사 분리, 더 명확한 병렬 추론/도구 사용을 주장하는 멀티스트림 LLM 논문을 발표했습니다.
*   δ-mem은 고정된 풀 어텐션 백본에 연결된 외부 온라인 연관 메모리를 제안했으며, 8x8 상태가 평균 점수를 1.10배 향상시키고 비-δ-mem 기준선을 1.15배 능가한다고 보고되었습니다.
*   NVIDIA의 Star Elastic은 한 번의 후처리 실행으로 추론 모델 크기 제품군을 도출할 수 있으며, 제품군을 사전 학습하는 것보다 360배 낮은 비용으로 SOTA 압축보다 7배 우수하다고 주장합니다.
*   OpenAI는 로컬 파일 시스템/도구 접근 권한으로 코딩 에이전트를 안전하게 실행하는 데 필요한 기술적 세부 정보를 설명하는 Windows 샌드박스 설계 보고서를 발표했습니다.
*   Kevin Li의 SWE-ZERO-12M-trajectories는 1120억 토큰, 1200만 트레이스, 12만 2천 개의 PR, 3천 개의 레포, 16개 언어를 포함하는 가장 큰 오픈 에이전틱 트레이스 데이터셋으로 포지셔닝되었습니다.
