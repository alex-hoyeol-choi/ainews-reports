*   **[SenseNova-U1, VAE/확산 없는 멀티모달 생성 모델 출시]** — SenseNova-U1이 Apache 2.0 라이선스로 출시되었으며, 2048x2048 해상도와 80억 파라미터를 가집니다. VAE나 확산 모델 없이 텍스트 렌더링을 이미지에 직접 통합하여 멀티모달 생성 및 이해를 수행하며, 인포그래픽 및 주석이 달린 다이어그램 생성에 강점을 보입니다.
*   **[GPT-5.5, 사이버 공격 시뮬레이션에서 인간 전문가 능가]** — GPT-5.5가 다단계 사이버 공격 시뮬레이션에서 Mythos를 능가하며, 인간 전문가에게 12시간 걸리는 작업을 1.73달러의 비용으로 11분 만에 완료했습니다. 이는 복잡한 사이버 보안 문제 처리에서의 효율성과 비용 효율성을 강조합니다.

### 🛠️ 제품 & 도구
*   xAI Grok 4.3이 Arena의 텍스트, 비전, 문서 및 코드 모드에 통합되었습니다.
*   OpenAI는 Codex 앱에 반응형 테스트를 위한 장치 툴바, 채팅 내 CI 상태, 설정/플러그인/에이전트용 마이그레이션/가져오기 도구, 그리고 펫 시스템을 추가했습니다.
*   Devin은 "쉘 내부" 핫키 액세스 기능을 출시했습니다.
*   Hermes는 에이전트가 완료될 때까지 계속하도록 강제하는 감독 모델이 있는 /goal 루프를 추가했습니다.

*   LangChain/LangGraph는 다중 사용자 및 human-in-the-loop 에이전트를 위한 데이터 격리, 위임된 자격 증명, 운영자 RBAC 문제를 해결하는 LangSmith Agent Server 기능을 선보였습니다.
*   LangChain은 인간의 응답이 도구 결과로 직접 반환될 수 있는 새로운 HITL(Human-in-the-Loop) 모드와 중요한 작업을 위한 내구성 있는 일시 중지/재개 시맨틱을 추가했습니다.
*   Cloudflare는 에이전트 계획에 내구성 있는 실행을 추가하기 위한 Dynamic Workflows를 발표했습니다.
*   LangChain의 `create_agent`는 Deep Agents 아래의 저수준 프리미티브로 자리매김하며, 파일 시스템, bash, 압축, 훅, 서브에이전트에 대한 확장성을 제공합니다.
*   PFlash는 C++/CUDA를 사용하여 양자화된 27B 모델의 긴 컨텍스트 디코딩을 위한 추측성 프리필 기법으로, RTX 3090에서 llama.cpp 대비 10배의 속도 향상을 달성했습니다.
*   Qwen 팀은 2B에서 35B MoE에 이르는 Qwen 3.5 모델을 위한 스파스 오토인코더(SAE) 세트인 Qwen-Scope를 출시했습니다. 이는 모델의 내부 개념 사전을 제공하여 특징 조작 및 디버깅을 가능하게 합니다.
*   Sulphur 2는 LTX-2.3 아키텍처를 활용한 오픈소스, 무검열 비디오 생성 모델로, 10초 길이, 24fps의 125,000개 비디오로 학습되었으며, Hugging Face에 출시될 예정입니다.
*   Z-Anime는 Alibaba의 Z-Image Base 아키텍처를 기반으로 애니메이션 스타일 이미지 생성을 위해 완전히 파인튜닝된 60억 파라미터 S3-DiT 모델입니다.
*   FLUX.2 Klein Identity Transfer Multi-Injection 도구는 대상 블록 내 여러 단계에서 참조를 주입하여 모델의 신원 전송을 향상시키도록 설계되었으며, 플러그 앤 플레이 프리셋으로 출시될 예정입니다.

### 🔬 연구 & 논문
*   ReaLM-Retrieve 논문은 추론 모델이 추론 중에도 리트리벌을 필요로 한다고 주장하며, 표준 RAG 대비 10.1%의 절대 F1 개선과 3.2배 낮은 리트리벌당 오버헤드를 보고했습니다.
*   OCR-Memory 논문은 장기 궤적을 인덱싱된 앵커가 있는 이미지로 저장하여, 손실이 있는 텍스트 요약 대신 정확한 이전 콘텐츠를 리트리벌하며, Mind2Web 및 AppWorld에서 SOTA를 달성했습니다.
*   Recursive Multi-Agent Systems 연구는 에이전트들이 자연어 교환 대신 공유된 잠재 재귀 계산을 통해 통신하는 방식을 제안하며, 평균 8.3%의 정확도 향상과 34.6%-75.6%의 토큰 감소를 보고했습니다.
*   Meta FAIR의 "자기 개선 사전 학습" 아이디어는 후처리 학습 모델이 사전 학습 접미사를 더 안전하고 고품질의 연속으로 다시 작성하고, RL 스타일 사전 학습 중에 모델 롤아웃을 판단하는 방법을 강조했습니다.
*   Microsoft는 현실적인 파일과 문서를 가진 1,000개의 합성 컴퓨터를 생성한 다음, 평균 2,000턴 이상의 8시간 에이전트 시뮬레이션을 실행하는 시스템을 개발하여, 컴퓨터 사용 에이전트의 병목 현상이 확장 가능하고 현실적인 경험적 데이터에 있음을 시사했습니다.
*   DeepSeek은 베이징 대학 및 칭화 대학과 협력하여 'Thinking with Visual Primitives' 멀티모달 추론 프레임워크를 도입했습니다. 이 프레임워크는 좌표점 및 바운딩 박스와 같은 공간 토큰을 모델의 CoT 프로세스에서 "최소 사고 단위"로 격상시킵니다.
*   OpenAI의 GPT-5.1은 '괴짜' 맥락에서 창의적인 언어에 보상을 주는 강화 학습 메커니즘 때문에 '고블린' 은유를 통합하기 시작했으며, OpenAI는 이후 학습 프로토콜을 조정하여 이 문제를 해결했습니다.
*   최신 1천만 편의 출판된 논문에 대한 인터랙티브 시맨틱 맵이 OpenAlex 데이터를 기반으로 출시되었습니다. SPECTER 2 임베딩과 UMAP을 사용하여 시맨틱 이웃을 형성하며, 키워드 및 시맨틱 쿼리를 지원합니다.
*   새로운 이미지 생성 기능에서 생성된 이미지가 시퀀스마다 품질이 저하되고 사실적 오류가 발생하는 현상이 보고되었습니다.
*   LLM은 불충분하게 명시된 요청을 오해할 가능성이 있어 정확하고 반복적인 작업 지침이 필요하다는 점이 강조되었습니다.
